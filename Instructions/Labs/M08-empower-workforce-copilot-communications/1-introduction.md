# 랩 설정

이 모듈에서는 다음 파일을 참조하는 Microsoft 365 Copilot용 프롬프트를 만들어 보겠습니다. 먼저 필요한 모든 파일을 OneDrive에 업로드하여 랩 전체에서 액세스할 수 있도록 하겠습니다.


### OneDrive에 파일 업로드

아래 단계에 따라 필요한 모든 파일을 **OneDrive**에 업로드합니다.

1. 테넌트 공급자가 제공한 가상 머신에 로컬 **관리자** 계정으로 암호를 사용하여 로그인합니다.`Pa55w.rd`
2. Windows 작업 표시줄에서 **Microsoft Edge**를 선택합니다.
3. 주소 표시줄에 `https://www.office.com`을 입력합니다.
4. **Microsoft 365 시작**에서 **로그인**을 선택합니다.
5. **로그인 프롬프트**에서 `userx@yourtenant.onmicrosoft.com`(테넌트가 제공한 사용자 이름과 테넌트) 입력 후 **다음**을 선택합니다.
6. **암호 입력** 화면에서 사용자 계정의 암호(테넌트 공급자가 제공)를 입력한 후 **로그인**을 선택합니다.
7. **로그인 유지**라는 메시지가 표시되면 **다시 표시하지 않음**을 선택한 다음 **예**를 선택합니다.
8. **Microsoft 365**에서 **앱**을 선택합니다.
9. **앱**에서 **OneDrive**를 선택합니다.
10. **OneDrive**의 왼쪽 상단에서 **+**(새로 추가) > **파일 업로드**를 선택합니다.
11. **파일 탐색기**에서 **내 PC** > **로컬 디스크(C:)** 를 선택하고 **ResourceFiles** 폴더를 엽니다.
12. **ResourceFiles** 폴더 내의 모든 파일을 선택한 다음 **열기**를 선택하여 **OneDrive**에 업로드합니다.
13. 업로드가 완료되면 화면 중앙 하단에 **내 파일에 29개 항목 업로드됨**이 표시됩니다.
14. **Edge**를 열어 두고 다음 작업으로 이동합니다.

### Copilot에서 파일 참조

Copilot을 사용할 때 일부 파일은 제안에서 즉시 사용할 수 없는 경우가 있습니다. 일부 Copilot 환경에서는 **최근에 사용한 목록**의 파일만 참조하는 반면, 다른 환경에서는 **OneDrive**를 직접 찾아볼 수 있기 때문에 이러한 현상이 발생합니다. 파일이 **최근에 사용한 목록**에 표시되도록 하려면 관련 Microsoft 365 앱에서 파일을 열기만 하면 자동으로 추가됩니다.

> [!IMPORTANT]
> Microsoft 365 Copilot은 **OneDrive**에 저장된 파일로만 작업할 수 있습니다. PC에 로컬로 저장된 파일은 Copilot에서 액세스하려면 **OneDrive**로 이동해야 합니다.

모듈을 진행하면서 이러한 파일에 대해 다양한 프롬프트를 시도해 볼 수 있습니다. Copilot 사용 실력을 향상시키기 위해 다양한 접근 방식으로 자유롭게 실험해 보세요.

# 소개
---
커뮤니케이션 전문가의 경우 명확하고 설득력 있고 시기 적절하게 메시지를 전달하는 기능은 내부적으로나 외부적으로 효과적인 참여의 핵심입니다. 오늘날의 빠르게 진행되는 콘텐츠가 풍부한 환경에서 커뮤니케이션 팀은 올바른 메시지를 작성할 뿐만 아니라 적절한 채널, 적절한 시간 및 적절한 대상 그룹에 전달되도록 하는 임무를 맡고 있습니다.

이 모듈에서는 Microsoft 365 Copilot의 기능을 소개하고 커뮤니케이션 전문가가 전략적 프롬프트를 사용하여 콘텐츠 생성을 간소화하고, 메시징 일관성을 향상시키고, 워크플로를 보다 효율적으로 관리하는 방법을 알아봅니다. 커뮤니케이션 전문가에게 Copilot 마스터는 단순히 유용한 기술이 아니라 경영진 연설 초안 작성에서 회사 발표 관리에 이르기까지 모든 것에 접근하는 방법을 재정의할 수 있는 혁신적인 변화입니다. Word에서 보도 자료를 편집하거나, SharePoint에서 캠페인 자산을 조정하거나, Outlook에서 내부 뉴스레터를 작성하거나, PowerPoint에서 리더십 브리핑을 준비하는 경우 Copilot는 커뮤니케이션 도구 키트의 강력한 확장이 됩니다.

이 모듈은 커뮤니케이션 전문가가 영향력을 높이는 데 필요한 기술과 사용 사례를 제공합니다. 커뮤니케이션 분야의 전문가로서, Microsoft 365 Copilot을 효과적으로 활용하는 능력은 다음과 같은 작업에 필수적입니다.

- **콘텐츠 만들기 간소화** Copilot은 보도 자료, 내부 메모, 리더십 메시지 등의 초안을 작성하는 데 도움을 줄 수 있습니다. 예를 들어, 어조를 다듬고 콘텐츠를 재구성하며 브랜드 보이스에 맞게 카피를 조정하는 데 도움을 줄 수 있습니다.
- **편집 워크플로를 개선합니다.** Copilot은 커뮤니케이션 전문가가 대용량 콘텐츠를 편집하고 검토할 수 있도록 지원합니다. 예를 들어, 긴 초안을 요약하고, 불일치하는 부분을 파악하고, 명확성과 어조를 개선할 것을 제안할 수 있습니다.
- **이해관계자 조율을 개선합니다.** Copilot을 사용하면 부서 간 원활한 협업이 가능합니다. 예를 들어 모임 또는 이메일 스레드에서 요약을 생성하여 메시징이 계획대로 진행되도록 유지하고 리더십 우선 순위에 부합하게 할 수 있습니다.
- **커뮤니케이션 계획 최적화** Copilot은 커뮤니케이션 팀이 복잡한 프로젝트를 구성하는 데 도움이 됩니다. 예를 들어 타임라인을 작성하고, 메시지 승인을 추적하고, 플랫폼 간에 게시 일정을 조정하는 데 도움이 될 수 있습니다.

Microsoft 365 Copilot은 지능형 AI 기반 협력자 역할을 합니다. 이를 통해 작성하는 모든 메시지에서 품질, 일관성 및 명확성을 유지관리할 수 있습니다. 이 사용 사례 모듈의 연습은 커뮤니케이션 전문가가 다음과 같이 Copilot의 잠재력을 최대한 활용할 수 있도록 설계되었습니다.

- **Word의 Microsoft 365 Copilot**을 사용하여 프로젝트 제안서 초안을 작성하고 구체화합니다.
- **Excel의 Microsoft 365 Copilot**을 사용하여 뉴스레터의 부서별 오픈율 및 클릭률을 분석하고 간단한 요약을 생성합니다.
- **Outlook의 Microsoft 365 Copilot**을 사용하여 내부 메일 공지를 작성, 편집 및 준비합니다.
- **Microsoft 365 Copilot Chat**Chat을 사용하여 회사 전체의 타운홀을 위한 주요 메시지를 컴파일하고 구체화합니다.

이 모듈을 마치면 Copilot를 사용하여 뛰어난 효율성, 영향 및 전략적 명확성으로 커뮤니케이션 함수를 이끌 준비가 되어 있어야 합니다.
